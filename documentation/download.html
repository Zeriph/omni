Library Downloads
Current library version: 0.8.4

Below you can download the latest source for the Omni library, as well, you can opt to download a copy of the examples, build or help files, or a combination of all.

Additionally, you can view the synced git repo, or if you have git installed, you can git clone https://github.com/Zeriph/omni.git

It should be noted if you download the help.zip file, a copy of this site is included as the HTML help documentation, but there is no Download link (since it links directly to the files you have already downloaded), so to cut on bloat and avoid confusion, it is omitted from the offline help documentation. To view the current source files without downloading the zip you can visit the source portal.

Also note that the raw_src.zip file contains the full commented source used to generate the help docs throughout this site. These source file are no different from the source files in src.zip except that the help documentation comments have been removed.
