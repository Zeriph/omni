OMNI_NO_CONSTS
MACRO

OMNI_NO_CONSTS - Disables any library defined constants.

SYNOPSIS top

Defining this macro will have the effect of having any library specific constants be undefined.

DESCRIPTION top

If this macro is enabled/defined, this will disable any constants defined in namespaces or classes. For example, instead of using the constant omni::math::PI in a funciton, you would have to utilize the OMNI_PI macro. This is in the rare event that the size of an executable should be taken into account; defining this compile time option will slightly reduce the overall size of the library code.

CONSIDERATIONS top

If this is macro is defined, any constants will be undefined and you must use their macro equivlents instead.

PLATFORM SPECIFIC top

No platform specific notes.

NOTES top

This does not disable any const values in local functions, it only disables the library constants.
