OMNI_OBJECT_NAME
MACRO

OMNI_OBJECT_NAME - If defined, adds a name string member to each object.

SYNOPSIS top

Defining this macro will add an extra member to each object.

DESCRIPTION top

If this macro is defined, then any classes in the library that can be instantiated (e.g. non-abstract classes and non-POD types) will have an omni::string_t member called name that can be used to name an object.
