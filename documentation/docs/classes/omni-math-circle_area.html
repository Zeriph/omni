SYNOPSIS

The mathematical constant e (Euler's number) as a long double.

top

DESCRIPTION

This is the mathematical constant e, also known as Euler's number. If OMNI_USE_M_E_L or OMNI_USE_MATH_DEFS are defined, and if the system has M_El defined, then this constant is set to M_El. Otherwise this constant is defined as 2.71828182845904523536028747135266250L
