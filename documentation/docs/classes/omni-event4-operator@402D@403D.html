NAME

omni::event4::operator-=

A member operator of omni::event4

#include <omni/delegate/4.hpp>
event4< Ret, PT1, PT2, PT3, PT4 >& omni::event4::operator-=(const omni::delegate4< Ret, PT1, PT2, PT3, PT4 >& d)


OVERLOADS

event4< Ret, PT1, PT2, PT3, PT4 >& omni::event4::operator-=(const event4< Ret, PT1, PT2, PT3, PT4 >& e)

SYNOPSIS

The -= operator is used to remove delegates from this event instance

top

DESCRIPTION

This is similar to calling the detach method on this instance.

top

RETURN VALUES

s A reference to this

top


PARAMETERS

d - The right most operand delegate to add

top


CONSIDERATIONS

If the OMNI_SAFE_EVENT defines are not enabled, it is undefined if this event is invoked and edited at the same time; i.e. a race condition can occur if you detach to an event from one thread while invoking from another.
